How to use:
1. Select an object.
2. In Texture Editor Menu -> Tools -> Align Horizontal or Align Vertical.
3. Switch On the button Island in TE and select an UV-island. Or can don't do it - just click RMB(right button mouse) to skip this step.
4. There is you should select an UV-edge. Switch Off the button Island. Select an edge.
5. Your texture projection or the island is rotated.
